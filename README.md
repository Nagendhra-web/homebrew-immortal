# homebrew-immortal

Homebrew tap for [Immortal](https://github.com/Nagendhra-web/Immortal) — the self-healing engine.

## Install

```bash
brew tap Nagendhra-web/immortal
brew install immortal
```

Or install directly:

```bash
brew install Nagendhra-web/immortal/immortal
```

## Usage

```bash
immortal start --watch-url https://myapp.com
```

See the [main repo](https://github.com/Nagendhra-web/Immortal) for full documentation.
